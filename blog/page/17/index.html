Eclipse安装及配置

Jun 9th, 2012 | Comments

官网下载：

  • oracle JDK
  • eclipse

1. 下载 java sdk - 7u4

linux平台:

  • x86-32: jdk-7u4-linux-i586.tar.gz
  • x64-64: jdk-7u4-linux-x64.tar.gz

windows平台:

  • x86-32: jdk-7u4-windows-i586.exe
  • x64-64: jdk-7u4-windows-x64.exe

2. 下载 eslipse

linux平台:

  • x86-32: eclipse-SDK-3.7.2-linux-gtk.tar.gz
  • x64-64: eclipse-SDK-3.7.2-linux-gtk-x86_64.tar.gz

windows平台:

  • x86-32: eclipse-SDK-3.7.2-win32.zip
  • x64-64: eclipse-SDK-3.7.2-win32-x86_64.zip

3. 安装

ubuntu 12.04 - 64bit

  1. 解压安装包到/opt路径下；
  2. 修改/opt/eclipse/eclipse.ini，在文件开头添加二行：

eclipse.ini:

-vm
/opt/jdk1.7.0_0x/bin/java

4. plugin - pydev

python的开发插件 - pydev，下载http://sourceforge.net/projects/pydev/files/pydev/ 的zip文件，解压后，对应复制features、plugins到eclipse，修改eclipse文件夹权限，否则仅root用户插件才起作用。
